# How to Run the Hybrid Smart Attendance System

## Prerequisites
Before you begin, make sure you have the following installed on your computer:
1.  **Python** (v3.8 or higher) - [Download Here](https://www.python.org/downloads/)
2.  **Node.js** (v18 or higher) - [Download Here](https://nodejs.org/en/download/)

---

## Step 1: Clean Up (Important)
If this folder was sent to you, it might contain system-specific files that won't work on your machine.
**Delete these folders if they exist:**
- `backend/venv`
- `frontend/node_modules`

---

## Step 2: Setup Backend (Python)
1.  Open a terminal (Command Prompt or PowerShell on Windows, Terminal on Mac).
2.  Navigate to the `backend` folder:
    ```bash
    cd path/to/this/folder/backend
    ```
3.  Create a virtual environment:
    - **Windows:** `python -m venv venv`
    - **Mac/Linux:** `python3 -m venv venv`
4.  Activate the virtual environment:
    - **Windows:** `venv\Scripts\activate`
    - **Mac/Linux:** `source venv/bin/activate`
5.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```
6.  Run the server:
    ```bash
    python app.py
    ```
    *The server should start on `http://localhost:5001`.*

---

## Step 3: Setup Frontend (React)
1.  Open a **new** terminal window.
2.  Navigate to the `frontend` folder:
    ```bash
    cd path/to/this/folder/frontend
    ```
3.  Install dependencies:
    ```bash
    npm install
    ```
4.  Run the application:
    ```bash
    npm run dev
    ```
    *The app should start on `http://localhost:5173`.*

## Troubleshooting
- **Port Conflicts:** If port 5001 or 5173 is busy, the terminal will tell you. You may need to kill the process or change the port in `app.py` / `vite.config.js`.
- **Database:** If you see "Table not found" errors, delete the `instance/attendance.db` file and restart the backend server.
